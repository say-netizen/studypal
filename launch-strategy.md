# StudyPal ローンチ戦略

**作成日**: 2026-04-28
**状況**: フォロワー0・マーケティング未着手・プロダクト完成済み
**URL**: https://studypal-chi.vercel.app

---

## 戦略の核心

> **「スタキャスで時間は測れる。StudyPalで成績が上がる。」**

スタキャスはすでに50万DLある。真正面から戦わない。
ターゲットユーザーが「スタキャスじゃ足りない」と感じる瞬間を狙う。
その瞬間は**テスト2週間前**。

---

## フェーズ計画（6週間）

### Phase 1：地盤固め（Week 1〜2）
*フォロワー0からでも今日始められること*

#### 1. Xアカウントを「勉強垢」として育てる
日本の中学生・高校受験生の最大集積地はX（Twitter）の「勉強垢」コミュニティ。

**アカウント設計**
- 名前: `StudyPal | AI学習アプリ` または `StudyPal_JP`
- プロフ: 「AI×カレンダーでテスト勉強を自動化。小5〜中3向け。無料で始める→[URL]」
- 固定ポスト: 30秒デモ動画 or スクリーンショット5枚スライドショー

**投稿戦略（毎日1〜2投稿）**
- 「#勉強垢と繋がりたい」「#中学生と繋がりたい」「#受験生と繋がりたい」を毎投稿に付ける
- 内容は「使い方Tips」「ユーザーの声」「機能紹介」ではなく**共感コンテンツ**から入る

**共感コンテンツ例**
```
テスト3日前「やばい何も勉強してない」
→ それ、2週間前から計画立ててたら防げてた

StudyPalはテスト日を入れるだけで
今日何を何分勉強すればいいか自動で出る

#中学生と繋がりたい #定期テスト対策
```

```
スタキャスで50分タイマー回した
でも何を勉強すればいいか結局わからなかった

「時間を測る」と「成績が上がる」は別の話

#勉強垢と繋がりたい
```

#### 2. Noteで保護者向けコンテンツを書く
保護者はXより**Note**にいる。SEO効果もある。

**記事テーマ（週1本）**
1. 「中学生の定期テスト、なぜ直前まで焦るのか？親にできる3つのこと」
2. 「スマホで勉強管理は危険？アプリ選びで成績が変わる理由」
3. 「AIで中学の勉強が変わる。我が子に使わせた感想」
4. 「保護者が子の勉強状況を把握できるアプリを比較してみた」

→ 記事末尾に毎回StudyPalへの誘導を入れる

#### 3. 初期ユーザー10人をDMで確保する
X・Instagramで「#定期テスト対策」「#中学生勉強」で検索し、投稿しているアカウントに個別DM。

**DMテンプレート**
```
はじめまして！中学生向けのAI勉強アプリ「StudyPal」を作っています。

テスト日程を入れるだけで今日の勉強計画が自動で作られる機能があって、
よかったら無料でベータ版使ってみてもらえませんか？

フィードバックもらえたらとても助かります🙏
```

### Phase 2：口コミの種まき（Week 3〜4）
*10人のユーザーを「伝道師」に変える*

#### 4. 勉強系YouTuberへのアプローチ
日本では「Study with me」「勉強法」系のYouTuberが中高生に強い影響力を持つ。

**ターゲット規模感**: チャンネル登録者1万〜10万人（大手より反応が来やすい）

**アプローチ文**
```
件名：【StudyPal】コラボのご相談

はじめまして、StudyPal開発者の[名前]と申します。

中学生向けのAI勉強計画アプリを作っており、
もし使っていただけるようであれば無料でご提供します。

テスト2週間前から計画を自動生成する機能が
チャンネルのフォロワーさんにも役立つのではと思い
ご連絡しました。
```

**探し方**: YouTube検索「中学生 勉強法」「定期テスト 勉強法」上位のチャンネル

#### 5. Instagram「勉強垢」への露出
Instagram での「#定期テスト」「#勉強垢」投稿者は中学生が多い。

**戦術**: リールで「30秒でわかるStudyPalの使い方」動画を投稿。
- 画面録画 + テロップ
- BGMはLofi系
- 「テスト2週間前にやること」シリーズとして複数本投稿

#### 6. ママ系コミュニティへの参加（保護者獲得）
- **Twitterの保護者垢**: 「#中学受験」「#中学生の親」でフォロー・リプ
- **ママスタ・Yahoo!知恵袋**: 「子供の勉強管理方法」スレッドに有益な回答 + アプリ紹介
- **LINE OpenChat**: 「中学生ママの会」等の公開グループに参加して情報提供

3. テスト後に結果を投稿 #StudyPalチャレンジ

#### 8. Product Huntローンチ（英語圏へのリーチ）
日本語サービスだが、Product Huntへの掲載で：
- 海外在住日本人への認知
- バックリンク獲得（SEO）
- 「PH掲載済み」の信頼性バッジ

**タイミング**: Week 6（準備が整ったタイミング）

#### 9. 塾・教師への直接アプローチ
**最も効果的な口コミ経路**: 塾の先生 → 生徒10〜30人に一気に広がる

**アプローチ先**
- 地域の個人塾（大手より動きが速い）
- 教育系Twitterアカウントを持つ先生

**提案内容**
```
先生のクラスの生徒全員に
StudyPal Familyプランを3ヶ月無料で提供します。

生徒の勉強状況を保護者も確認でき、
授業の復習計画も自動で立てられます。
```

## チャネル戦略まとめ（ORBフレームワーク）

### Owned（積み上げ資産）
| チャネル | 用途 | 優先度 |
|---------|------|--------|
| メールリスト | サインアップ後のオンボーディング・ニュースレター | ★★★ |
| Note.com | 保護者向けSEOコンテンツ | ★★★ |
| アプリ内（StudyPal本体） | ウイルス性機能（ランキング共有・招待） | ★★★ |

### Rented（プラットフォーム依存）
| チャネル | ターゲット | 戦術 |
|---------|-----------|------|
| X (Twitter) | 中学生・受験生 | #勉強垢コミュニティ、共感コンテンツ |
| Instagram | 中学生（女子中心） | リール動画、#定期テスト |
| TikTok | 10代全般 | 30秒Tips動画 |
| YouTube | 中高生・保護者 | Study with me系との連携 |

### Borrowed（他者の集客力を借りる）
| 対象 | 規模感 | アプローチ |
|------|--------|-----------|
| 勉強系YouTuber | 1万〜10万登録 | 無料提供DM |
| 教育系Xインフルエンサー | 1000〜5万フォロワー | コラボ投稿依頼 |
| 個人塾の先生 | クラス10〜30人 | 無料提供・クラス導入 |
| 保護者ブロガー（Note） | 読者1000〜 | 体験記事依頼 |

## メッセージング戦略

### ユーザー別メッセージ

**生徒向け（小5〜中3）**
> 「テスト前に焦らなくなる。AIが今日の勉強計画を作ってくれる。」

**保護者向け**
> 「子供の勉強状況が、スマホで確認できる。テスト2週間前にアラートが届く。」

**塾・先生向け**
> 「生徒全員の勉強記録と目標達成率が一覧で見られる。保護者との連絡もスムーズに。」

### スタキャスとの差別化メッセージ
| スタキャス | StudyPal |
|-----------|---------|
| 時間を測る | 成績を上げる計画を立てる |
| 「今日何分勉強した」 | 「今日何を何分勉強すればいいか」 |
| 友達と繋がれる | 親子で一緒に目標を持てる |
| 無料だけど何も変わらない | AI + 計画で本当に点数が上がる |
